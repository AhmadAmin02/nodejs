Apa itu REST API?
REST API (Representational State Transfer Application Programming Interface) adalah arsitektur komunikasi berbasis HTTP yang memungkinkan aplikasi atau sistem untuk saling berkomunikasi melalui internet dengan prinsip REST. REST adalah gaya arsitektur yang menggunakan metode standar HTTP seperti GET, POST, PUT, PATCH, dan DELETE untuk melakukan operasi terhadap sumber daya (resources).

Client dan server dipisahkan sehingga client hanya meminta data, sedangkan server menyediakan data tersebut tanpa ketergantungan satu sama lain. REST API bersifat stateless, yang berarti setiap permintaan dari client harus mengandung semua informasi yang diperlukan karena server tidak menyimpan sesi atau state dari permintaan sebelumnya.

Data dalam REST API biasanya dikirim dalam format JSON atau XML, dengan JSON yang lebih umum digunakan karena lebih ringan dan mudah dibaca oleh manusia maupun mesin. REST API juga mendukung caching untuk meningkatkan performa dengan menyimpan respons agar tidak perlu memproses ulang permintaan yang sama.

REST API banyak digunakan untuk membangun sistem berbasis web, termasuk layanan backend untuk aplikasi mobile dan web, karena fleksibel, mudah diimplementasikan, dan dapat digunakan oleh berbagai platform dengan protokol HTTP standar.
